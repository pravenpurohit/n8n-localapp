# Screen 05: Global Executions Page

## Phase: 1 (Free Community Edition)

## Description
Shows all workflow executions across the entire n8n instance. Accessible from the sidebar or the Overview > Executions tab. Provides filtering, detail views, retry, and delete capabilities.
